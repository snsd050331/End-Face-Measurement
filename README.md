# 📏端面精準量測 - 檢測鋼胚對角線數據
## 安裝依賴項
首先先安裝Yolo相關套件  
在 [**Python>=3.8**](https://www.python.org/) 環境中安裝 `ultralytics` 包，包括所有[依賴項](https://github.com/ultralytics/ultralytics/blob/main/pyproject.toml)，並確保 [**PyTorch>=1.8**](https://pytorch.org/get-started/locally/)。
